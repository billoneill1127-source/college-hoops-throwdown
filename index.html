Version History

v2.4
Announcement engine overhaul. And-one plays now produce a single combined basket+foul announcement instead of two separate lines. Run milestone announcements include score context (leading, tied, or cutting the deficit). Run announcements now always follow the basket line in the play-by-play. Run counter saved and restored in snapshots, reset at halftime; thresholds raised to 8/10/12/14. Press/trap fouls no longer route through the intentional foul path. Duplicate CPU defense-change announcements suppressed via snapshot-restored lastLoggedDefense and point-of-change logging. In-game substitution screen reads stats from the current display-position snapshot instead of end-of-simulation player object values, fixing stale foul/minute counts.

v2.3
Coaching mode. Interactive lineup management with foul-out interrupt — P1 selects replacement when a player fouls out. Strategy overlay shows post-substitution lineup. Gameplan page removed; coaching mode toggle moved to lineup screen. Pre-game columns show season averages; in-game columns show live MIN/PTS/REB/AST/PF/FG/FT. Two-player guardrail on Pick & Roll / Pick & Pop dropdowns. Snapshot-based resim change detection prevents lineup sync from suppressing resimulation.

v2.2
Modularized architecture. Strategy system extracted from index.html into shared module strategy.js (window.Strategy). Strategy modifier table, CPU strategy selection, and random assignment for sim games now shared across index.html, season.html, and simulate.html. game_engine_sim.js applies live strategy mods per possession when strategy.js is loaded; falls back to neutral mods when running standalone.

v2.1
CPU rotation substitution system. Players sub in/out based on minutes budget (MPG × 0.92) and foul trouble thresholds (3 fouls in first half, 4 in second). Rotation players (MPG ≥ 8) managed separately from emergency depth. Minutes played now tracked via possession counter (poss) rather than static MPG. Net rating quality-of-competition adjustments applied across shooting, steals, turnovers, blocks, and rebounds.

v2.0
Tournament mode. 64-team single-elimination bracket seeded from net ratings (tiers A–D with conference-conflict resolution). Full bracket UI with regional tabs (East, Southeast, Midwest, West) and SVG connector lines. CPU games simulate automatically with animated score reveals; user plays their games through the normal game engine with neutral-site rules. Box score shown after every user game. Bracket persists in localStorage; sim the rest at any time after elimination. Fixed Final Four / Championship PLAY button (cross-region prerequisite check). Net rating quality-of-competition adjustments applied across shooting, steals, turnovers, blocks, and rebounds.

v1.5
Block system (checkBlock / resolveBlockedShot / resolvePutbackOrPossession) wired into runPossession with 9 randomised announce templates and momentum swing on clean blocks. Assist crediting moved post-make, driven by team assist_rate; removed pre-shot assister logic. Real team stats (pace, ORB%, DRB%, AST%, fouls/game) loaded from Sports Reference via 3 extract scripts and apply_team_stats.js pipeline. Pace calibration confirmed: 68 base possessions per team vs real mean 68.6. Non-shooting foul retry no longer inflates possession count. Blk column added to simulate.html.

v1.4
Fixed simulate.html team dropdowns (corrupted template literals replaced with Option constructor). Version bump across all pages.

v1.3
Added 12 new independent teams. Fixed gameId collisions on simultaneous simulations. Rebuilt simulate.html: replaced stale 850KB inline DATA blob with teams-data.js. Fixed GameEngineSim double-nesting bug. Coordinated non-conf scheduler prevents same opponent appearing on same date. Removed simulatedThisRun dedup. Independent teams hidden from season setup conference picker.

v1.2
Split non-conf schedule into 6 P5 games and 6 Independent games (3H/3A each). Added independent teams to data pipeline via migrate.js auto-scan. Fixed teams-data.js to expose window.CONFERENCES alongside window.TEAMS.

v1.0
Initial release. Season mode, exhibition play, stats dashboard, conference schedules, box score logging.
